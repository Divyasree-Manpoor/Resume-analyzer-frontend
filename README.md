# AI Resume Analyzer Frontend

A modern and responsive React.js frontend application for the AI Resume Analyzer platform. This application provides an intuitive user interface for resume analysis, ATS score evaluation, job description matching, career roadmap generation, and AI-powered career assistance.

---

## Features

### Resume Analysis

* Upload resumes in supported formats
* AI-powered resume evaluation
* Resume quality insights and recommendations

### ATS Score Checker

* Analyze resume compatibility with Applicant Tracking Systems (ATS)
* ATS score visualization
* Improvement suggestions

### Job Description Match

* Compare resumes with job descriptions
* Match percentage calculation
* Skill gap identification

### Career Roadmap

* Personalized career growth recommendations
* Skill development guidance
* Learning path suggestions

### AI Career Assistant

* AI-powered career guidance
* Resume improvement recommendations
* Career-related question support

### User Experience

* Fully responsive design
* Mobile, tablet, and desktop support
* Dark mode and light mode
* Smooth animations and transitions
* Modern UI with Tailwind CSS and Framer Motion

## Installation

### Clone Repository

```bash
git clone <frontend-repository-url>
cd ai-resume-analyzer-frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Create Production Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

---

## Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=http://localhost:5000/api
```
